# aex-bot

### Installation Steps

1. `$ cd aex-bot`
1. `$ npm i`
2. `$ npm start`

``Note : runing at port 3000``

### Usage
<pre>
After scan barcode whatsapp, you can stat message with following command : !help for showing menu/command
</pre>
